NVM 安装

NVM 全称 Node Version Manager，顾名思义 Node.js 版本管理工具

本人用的Mac，直接用 Homebrew 安装 NVM

根据安装提示创建目录，导入环境变量后记得 source，nvm -v 检查是否安装成功

nvm 使用方法 nvm -h，学过英文一般也看懂了

安装 Node.js

使用 nvm 安装 node.js，这里安装最新的 LTS 版本

node -v 检查版本
