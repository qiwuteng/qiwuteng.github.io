Ubuntu 特效动画关闭和打开 Ubuntu 桌面带有一定的特效动画，若使用虚拟机会显得卡顿，若想关闭特效，就在终端输入： 1gsettings set org.gnome.desktop.interface enable-animations false 该命令会将 GNOME 默认的画面切换的动画功能关闭。 若想重新打开特效，输入以下命令： 1gsettings set org.gnome.desktop.interface enable-animations true
